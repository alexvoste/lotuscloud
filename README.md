## 📋 О программе

Департамент инфраструктуры Lotus Cloud объявляет о запуске программы интеграции внешних вычислительных узлов (Residential Nodes) в распределённую сеть.

Партнёры получают возможность монетизировать доступные ресурсы сервера и получать вознаграждение в LC токенах.

## ⚡️ Преимущества

- 💰 **Монетизация** — вознаграждение за аптайм и трафик
- 🔒 **Безопасность** — изолированная среда Docker
- ⚙️ **Автоматизация** — развёртывание через Docker
- 🖥️ **Минимальные требования** — достаточно базового железа

## 🖥️ Требования к системе

### Аппаратные требования

| Параметр | Минимум      |
| -------- | ------------ |
| CPU      | 1 ядро       |
| RAM      | 2 GB         |
| SSD      | 10 GB        |
| Канал    | от 50 Мбит/с |

### Сетевые требования

| Тип IP                | Поддержка            |
| --------------------- | -------------------- |
| Статический белый IP  | ✅ Полная            |
| Динамический белый IP | ⚠️ Ограниченная      |
| Серый IP (NAT)        | ⚠️ Требует настройки |

> **Проверка IP-адреса**

```bash
curl ipconfig.me
```

> Перезагрузите роутер и повторите. Если адрес не изменился — у вас статический IP.

### Поддерживаемые ОС

- Ubuntu 22.04 (рекомендуется)
- Debian 12
- Arch Linux

## 🚀 Установка

### Linux

1. **Клонируйте репозиторий:**

```bash
git clone https://github.com/alexvoste/lotuscloud
cd lotuscloud
```

2. **Запустите скрипт установки:**

```bash
chmod +x install.sh
./install.sh
```

Скрипт автоматически:

- Установит Docker
- Установит необходимые пакеты
- Настроит docker-compose.yml

### Windows

Инструкция доступна в [Wiki](https://github.com/alexvoste/lotuscloud/wiki/Node%E2%80%90Setup).

## 🔧 Архитектура безопасности

Узлы развёртываются в изолированной среде Docker через **Marzban-node**. Доступ к основной системе не требуется.

```bash
┌─────────────────────────────────────┐
│         Основная система            │
├─────────────────────────────────────┤
│  ┌───────────────────────────────┐  │
│  │     Docker (изолированная)    │  │
│  ├───────────────────────────────┤  │
│  │    ┌─────────────────────┐    │  │
│  │    │   Marzban-node      │    │  │
│  │    └─────────────────────┘    │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```
